Meek Warrior

WordPress has been around for a long time and has been through many iterations, becoming better with each new version. But it still has its problems. Open source programs are a great concept but there are many compromises on what should be considered as essential features and on security issues. Specific complaints include the endless updates and constantly changing interface and the fact that it is incomplete. We need plugins for everything from SEO to e-commerce because WordPress does not include commonly needed features in the core program.
